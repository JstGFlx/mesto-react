# Mesto - React

## Описание

Полностью переписанный на React проект [Mesto](https://github.com/JstGFlx/mesto)

## Технологии использующиеся в проекте и функции
- стек HTML, CSS, JavaScript, React
- Реализована работа с серверным api
- Ассинхронный код
- Сборка WebPack
- JS модули
- ООП парадигма в написании кода
- Динамическое создание карточек
- Интерактивный попап редактирования профиля
- Интерактивный попап добавления новых карточек
- Адаптивная верстка

## Установка и развертываение:
Клонируйте репозиторий командой:
```
git clone https://github.com/JstGFlx/mesto-react.git
```
Далее потребуется установленный node.js и его пакетный менеджер npm, найти их сможете тут: http://nodejs.org/
После установки npm перейдите в папку с проектом и введите команду:

```
npm i
```
Установятся все необходимые зависимости. После этого введите команду:
```
npm start
```
Проект станет доступен по адресу: http://localhost:3000/
